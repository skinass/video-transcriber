# Video Transcriber

Инструмент для автоматической транскрибации видеофайлов с помощью [OpenAI Whisper](https://github.com/openai/whisper).

- Пакетная обработка — транскрибирует все новые видео из папки, пропуская уже готовые
- Поддержка GPU — автоматически использует NVIDIA GPU (CUDA), если доступен
- Умные промпты — для видео с CS:GO в названии применяется расширенный промпт с игровым сленгом

## Быстрый старт

1. Положи видеофайлы в папку `raw/`
2. Запусти `run.bat`

При первом запуске автоматически создаётся виртуальное окружение и устанавливаются все зависимости. Готовые транскрипции появятся в папке `subtitles/`.

## Конфигурация

Настройки хранятся в `transcribe.yaml`:

| Параметр | По умолчанию | Описание |
|---|---|---|
| `source` | `raw` | Папка с входными видеофайлами |
| `destination` | `subtitles` | Папка для выходных `.txt` файлов |
| `model` | `small` | Размер модели Whisper (см. ниже) |
| `language` | `ru` | Язык аудио (`null` — автоопределение) |

### Размеры моделей

| Модель | Скорость | Точность | VRAM |
|---|---|---|---|
| `tiny` | максимальная | низкая | ~1 GB |
| `base` | быстрая | приемлемая | ~1 GB |
| `small` | баланс | хорошая | ~2 GB |
| `medium` | медленная | высокая | ~5 GB |
| `large` | очень медленная | максимальная | ~10 GB |

## Требования

- Python 3.8+
- ffmpeg (устанавливается автоматически через `imageio-ffmpeg`)
- NVIDIA GPU с CUDA 12.8+ — опционально, без него работает на CPU
